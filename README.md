# Java_GUI_Push-model
Java의 GUI와 소켓 통신을 통해 간단한 채팅 프로그램을 만들어 보았습니다.

<br>

## 프로젝트 목표
소켓 통신을 통해 서버와 클라이언트를 직접 만들어 서로 통신 하는 것을 구현해본다.

<br>

## 구현 기능
- 서버기능
  - 서버는 여러 클라이언트를 접속 허용한다.
- 클라이언트 기능
  - 클라이언트 서버가 열려 있다면 접속할 수 있다.
  - 클라이언트끼리 서로의 채팅을 볼 수 있다.

<br>
